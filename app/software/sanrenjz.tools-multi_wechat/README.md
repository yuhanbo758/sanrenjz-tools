# 微信多开

版本：1.0.0 | 作者：三人聚智 | 分类：系统工具

## 简介

电脑端微信多开工具，选择微信程序路径后一键启动多个独立微信实例，满足多账号同时在线需求。

> 仅支持 Windows 系统

## 主要功能

- **一键多开**：点击按钮即可再次启动一个新的微信实例
- **自定义路径**：支持手动指定微信安装路径，适配自定义安装位置
- **自动检测**：自动扫描默认安装路径，免配置直接使用
- **无限实例**：可重复点击启动任意数量的微信实例（受内存限制）

## 使用方法

1. **打开插件**：搜索框输入「微信多开」→ 回车打开
2. **自动检测**：插件会自动扫描常见微信安装路径
3. **手动指定**（如自动检测失败）：点击「选择路径」→ 找到 `WeChat.exe` 所在目录
4. **启动多开**：点击「启动微信」按钮，即可再开一个微信实例
5. **超级面板**：右键长按 → 选择「微信多开」快速启动

## 微信常见安装路径

```
C:\Program Files\Tencent\WeChat\WeChat.exe
C:\Program Files (x86)\Tencent\WeChat\WeChat.exe
D:\Program Files\Tencent\WeChat\WeChat.exe
```

## 注意事项

- **仅支持 Windows**，macOS 版微信官方支持多开，无需此工具
- 多开微信会占用更多内存，请确保系统内存充足
- 本工具通过直接启动 `WeChat.exe` 实现多开，不修改微信程序文件
- 如微信更新后无法多开，可能需要重新指定路径
- 部分企业微信安全策略可能限制多开行为
