# 余汉波AI助手

版本：1.4.0 | 作者：余汉波 | 分类：效率工具

## 简介

多模型 AI 对话助手，支持 OpenAI、DeepSeek、Gemini、通义千问等主流大语言模型 API。可直接在工具平台内进行对话，也支持选中任意文字后一键发送 AI 分析。

## 主要功能

- **多模型切换**：在设置页配置多个模型 API Key，随时切换使用
- **选中文本分析**：在任意应用中选中文本，通过超级面板或搜索框快速发送给 AI
- **快捷指令**：预设常用提示词（快捷指令），一键触发，无需每次重复输入
- **对话历史**：保留当前会话上下文，支持多轮对话
- **Markdown 渲染**：AI 回复支持 Markdown 格式展示（代码块、表格、列表等）

## 使用方法

1. **首次使用**：打开插件 → 进入设置 → 填写大模型 API Key 和接口地址
2. **普通对话**：搜索框输入 `AI助手` 或 `GPT` 打开插件，在输入框输入问题后回车
3. **选中文本分析**：在任意位置选中文字 → 呼出搜索框（Ctrl+Space）→ 选择「AI助手」，所选文字会自动填入输入框
4. **超级面板**：右键长按 → 选择「AI助手」快捷入口
5. **快捷指令**：在设置页添加常用提示词，可在对话界面一键选用

## 支持的模型

| 模型 | API 提供方 |
|------|-----------|
| GPT-4 / GPT-3.5 | OpenAI |
| DeepSeek-V3 / R1 | DeepSeek / 硅基流动 |
| Gemini Pro | Google |
| Qwen | 阿里云通义千问 |
| 其他兼容 OpenAI 接口的模型 | 自定义 Base URL |

## 注意事项

- API Key 保存在本地，不会上传到任何服务器
- 需要网络访问对应的 API 接口，请确保网络畅通
- 选中文本最大支持 50,000 字符
