# 图片裁剪工具

一个功能强大的图片裁剪和编辑插件，专为三人聚智工具平台开发。支持拖拽上传、智能裁剪、文字添加、多格式导出等功能。

## ✨ 主要功能

### 📷 图片处理
- **拖拽上传**：支持直接拖拽图片文件到界面
- **文件选择**：点击按钮选择本地图片文件
- **格式支持**：JPG、PNG、WebP、GIF、BMP等主流格式
- **图片旋转**：支持左转90°和右转90°
- **一键重置**：快速恢复到原始状态

### ✂️ 智能裁剪
- **预设尺寸**：提供多种常用尺寸比例
  - 1:1 正方形（头像、Logo）
  - 16:9 横屏（视频封面）
  - 9:16 竖屏（手机壁纸）
  - 4:3 传统比例
  - 3:4 竖封面比例
  - 21:9 超宽屏
- **自定义尺寸**：手动输入宽度和高度
- **自由裁剪**：不限制比例的自由裁剪
- **实时预览**：所见即所得的裁剪效果

### 🎨 文字编辑
- **添加文字**：在图片上添加自定义文字
- **字体选择**：支持系统字体（微软雅黑、宋体、黑体等）
- **字号调整**：8-200px字号范围
- **颜色设置**：自定义文字颜色
- **透明度控制**：0-100%透明度调节
- **拖拽移动**：鼠标拖拽调整文字位置
- **实时编辑**：选中文字后可实时修改内容和样式

### 💾 导出功能
- **多格式支持**：PNG、JPG、WebP格式导出
- **质量控制**：10-100%质量调节（JPG/WebP）
- **一键下载**：自动生成文件名并下载
- **高清输出**：保持原始图片质量

## 🚀 使用方法

### 基础操作
1. **加载图片**
   - 方法一：直接拖拽图片文件到上传区域
   - 方法二：点击"选择图片"按钮选择文件
   - 方法三：点击工具栏的"📁 选择图片"按钮

2. **图片调整**
   - 点击"↺ 左转"或"↻ 右转"旋转图片
   - 点击"🔄 重置"恢复原始状态

### 裁剪操作
1. **选择裁剪尺寸**
   - 在左侧面板选择预设尺寸比例
   - 或选择"自定义"手动输入宽高
   - 或选择"自由裁剪"不限制比例

2. **执行裁剪**
   - 点击工具栏的"✂️ 裁剪"按钮
   - 系统会自动按照设定尺寸进行裁剪

### 文字编辑
1. **添加文字**
   - 点击"添加文字"按钮
   - 在画布中央会出现默认文字

2. **编辑文字**
   - 点击文字元素选中
   - 在文字工具面板中修改内容、字体、大小、颜色等
   - 拖拽文字到合适位置

3. **删除文字**
   - 选中要删除的文字元素
   - 点击"删除选中文字"按钮

### 导出图片
1. **选择格式**
   - 在导出设置中选择PNG、JPG或WebP格式
   - 调整图片质量（仅JPG/WebP格式）

2. **下载图片**
   - 点击"下载图片"按钮
   - 系统会自动生成文件名并开始下载

## 💡 使用技巧

1. **批量处理**：可以重复使用相同的裁剪设置处理多张图片
2. **文字设计**：合理使用字体大小和颜色，确保文字清晰可读
3. **格式选择**：
   - PNG：支持透明背景，适合Logo和图标
   - JPG：文件较小，适合照片和复杂图像
   - WebP：新格式，体积小质量高
4. **质量设置**：一般建议JPG质量设为80-90%，平衡文件大小和画质
