# 余汉波文本片段助手

版本：1.0.0 | 作者：余汉波 | 分类：效率工具

## 简介

快速搜索 Markdown 文件并将内容直接插入当前输入框的文本片段管理工具。将常用的文本、模板、代码片段保存为 Markdown 文件，通过关键词秒速检索、一键插入。

## 主要功能

- **片段管理**：以 Markdown 文件为单位管理文本片段，支持多级目录分类
- **快速搜索**：输入关键词模糊匹配片段标题或内容，毫秒级响应
- **一键插入**：选中片段后自动将内容插入当前光标所在输入框
- **关键字上屏**：为片段设置触发关键字，输入关键字后自动替换为对应内容
- **超级面板集成**：右键长按可快速访问文本片段列表
- **路径配置**：可自定义 Markdown 文件存放目录

## 使用方法

1. **配置路径**：搜索框输入「文本片段设置」→ 配置 Markdown 文件所在目录
2. **创建片段**：在配置的目录下创建 `.md` 文件，文件名即为片段名
3. **搜索插入**：
   - 搜索框输入「文本片段」打开管理界面
   - 输入关键词筛选，方向键选择，Enter 插入到当前应用
4. **关键字上屏**：在片段文件顶部按约定格式设置关键字，输入关键字后自动展开
5. **超级面板快速访问**：右键长按 → 「文本片段快速访问」→ 选择并插入

## 片段文件格式

片段为普通 Markdown 文件（`.md`），文件名作为搜索标题，文件内容即插入内容。建议按分类创建子目录管理片段。

## 注意事项

- 片段目录下的所有 `.md` 文件均会被索引
- 插入功能依赖模拟键盘操作，部分应用（如管理员权限程序）可能无法自动插入，需手动粘贴
- 首次使用需先在设置中配置存储路径
