# AI 屏幕翻译

版本：1.0.1 | 作者：余汉波 | 分类：AI 工具

## 简介

基于硅基流动 DeepSeek 模型的文本与截图翻译插件，支持中英互译与 OCR 识别。复制文字或截图后一键翻译，无需切换浏览器。

## 主要功能

- **文本翻译**：选中任意文字或复制文字到剪贴板，自动进行中英互译
- **截图翻译（OCR）**：复制图片或截图到剪贴板，自动识别图中文字并翻译
- **超级面板集成**：右键长按后可直接触发翻译，无缝嵌入工作流
- **中英互译**：自动检测语言方向，中文翻英文、英文翻中文

## 使用方法

### 文本翻译

1. 选中需要翻译的文字（或复制到剪贴板）
2. 呼出搜索框（Ctrl+Space）→ 输入「翻译」或「DeepSeek」
3. 选择「DeepSeek 文本翻译」→ 翻译结果自动显示
4. 或：右键长按 → 超级面板中选择「DeepSeek 翻译」

### 截图翻译

1. 使用截图工具截图（复制到剪贴板），或复制图片文件
2. 呼出搜索框 → 选择「DeepSeek 截图翻译」
3. 或：右键长按 → 超级面板中选择「DeepSeek 截图翻译」
4. 插件自动 OCR 识别图中文字并完成翻译

## 配置说明

首次使用需在插件设置页填写**硅基流动 API Key**：

1. 注册 [硅基流动](https://siliconflow.cn) 并获取 API Key
2. 打开插件 → 进入设置 → 填写 API Key → 保存

## 注意事项

- 翻译功能需要网络连接访问硅基流动 API
- 文本翻译最大支持 10,000 字符
- OCR 截图翻译效果取决于图片清晰度
- API Key 保存在本地，不会上传
